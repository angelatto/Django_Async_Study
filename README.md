# Django_Async_Study

- RabbitMQ, Redis 설치 및 로컬 세팅 완료  
- 로컬에서 RabbitMQ에 비동기task 적재 실습 완료 
- 로컬에서 celery worker 띄우고 task 실행하는 실습 완료
- celery worker가 비동기 작업 후 결과를 redis에 저장 실습 완료 
- local redis에 결과 저장 및 키값 조회, 삭제 등 기본명령어 실습 완료
